# OpenMP及MPI并行程序设计

使用OpenMP及MPI完成的几个并行程序设计小实验：矩阵相乘、矩阵LU分解、文档分类中的文档向量过程



**我所使用的运行环境**

+ Ubuntu 16.04（64位版本）

+ gcc version 5.4.0 20160609 （低版本gcc可能无法使用OpenMP）

+ mpicc for MPICH version 3.3.2（请根据网上教程安装好MPI环境）
